# Постановка: упаковка промптов — blog + Telegram + LinkedIn

## Цель (одно предложение)
Довести почти готовый материал об упаковке промптов (язык / форма / место) до публикации на
tellian.io (лонгрид) + Telegram (пост) + LinkedIn (пост), с проверенными источниками.

## Что уже есть (в materials/)
- `prompt-upakovka.md` — отполированный RU-лонгрид (почти финал).
- `prompt-upakovka-telegram.md` — RU-пост для телеги (опенер слишком технический).
- LinkedIn — **нет, делаем с нуля.**
- `PROJECT-CONTEXT.md` — ресёрч-лог, аннотированная библиография, план, TODO, развилки.

## Главный риск / где основная работа
Библиография из ИИ-чата: часть URL «уточнить», много arxiv-ID 2026 без проверки. По харнесу —
**каждое несущее утверждение → реальный источник + цитата.** Фактчек = ядро задачи; что не
разрешается → режем или метим `[UNVERIFIED]`.

## Ship-bar
- [ ] Каждое несущее утверждение и каждая цифра разрешены в реальный источник (или помечены)
- [ ] Тезис ясен в первом экране; мастер-таблица на месте
- [ ] Три канала готовы; деривативы согласованы с каноном
- [ ] Прожарка (editor) + mirror-editor + reader-fan пройдены
- [ ] Явный гейт Макса перед публикацией

## DO NOT
- Не выдавать непроверенную ссылку за источник; не «дорисовывать» arxiv-ID.
- Не превращать в вирусный листикл; держать честные цифры с базой.
- Не ломать авторский голос Макса без причины (latitude — см. вопросы).
