## Что сделано

- **Фактчек:** каждое опорное утверждение → резолвящийся источник; непроверенные числа выкинуты
  (напр. «русский ×3» → проверенный ×2, «47%» убран).
- **Прожарка ×4:** editor (REVISE→исправлено), mirror (SHIP), reader-fan («поделился бы»),
  stylist-ru/en. Дедуп по-модельной секции + сжатие TOON, builder-глубина цела.
- **Форматинг по правкам Макса:** карточки-выноски, TL;DR, эмодзи-подзаголовки, «язык пользователя».
- **Деривативы:** Telegram (RU, hook-first), LinkedIn (EN, инсайт-хук) — оба линкуются на блог.
- **Публикация:** через `wp_publish.py` (билингва-пост, CSS-свитчер), `wp_post_id 260`.
